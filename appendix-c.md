
# Appendix C — Glossary

*Every term in WinUbuntu, gathered and plainly defined. (Mission where it first appears in parentheses.)*

---

**apt** *(M2)* — Ubuntu's package manager: its "app store," used from the terminal. Installs software cleanly and reversibly.

**AI Landscape** *(M0)* — the seven-level map of the AI world, from browser AI up to cloud AI.

**Build loop** *(M7)* — the repeating rhythm of building: *write → run → see → improve.*

**cat** *(M4)* — a command that prints a file's contents to the screen.

**cd** *(M4)* — "change directory": move into another folder. `cd ~` goes home.

**Clone** *(M6)* — make a full local copy of a GitHub repository with `git clone`.

**Cloud / GPU machine** *(M8)* — a more powerful computer you rent remotely and reach with the same commands.

**Command** *(M1)* — a short instruction you type into the terminal and run (like `whoami`).

**curl** *(M3)* — a tool that downloads things from the internet in the terminal.

**Directory / folder** *(M4)* — a container for files (and other folders).

**Elephant in the Room** *(M0)* — the practice of naming a fear out loud so curiosity has room to grow.

**Git** *(M6)* — a tool that tracks every version of your files, so nothing is ever truly lost.

**GitHub** *(M6)* — the website that hosts Git projects and lets people share them.

**Home (`~`)** *(M4)* — your personal folder, e.g. `/home/wendy`. `cd ~` always returns you here.

**history** *(M8)* — a command listing every command you've run.

**import** *(M7)* — in Python, brings an extra tool (module) into your script.

**input()** *(M5)* — in Python, asks the user a question and waits for their answer.

**Integration** *(M7)* — wiring separate tools together into one thing (e.g., Python + your local AI).

**Interpreter (`python3`)** *(M5)* — the program that reads and runs your Python scripts.

**Local AI** *(M3)* — AI that runs on your own machine, offline and private, not on a website.

**ls** *(M4)* — "list": show what's in the current folder.

**mkdir** *(M4)* — "make directory": create a new folder.

**Model** *(M3)* — the file of learned patterns that "is" an AI. Small models run on ordinary laptops.

**nano** *(M4)* — the beginner-friendly text editor built into Ubuntu. Save: **Ctrl+O**, Enter; exit: **Ctrl+X**.

**Ollama** *(M3)* — a tool that downloads and runs AI models locally, and lets you chat with them.

**Package** *(M2)* — a piece of software that `apt` can install.

**Path** *(M4)* — a folder's full address, like `/home/wendy/winubuntu`.

**print()** *(M5)* — in Python, shows text on the screen.

**Prompt** *(M3)* — the message you send to an AI model. *(Also: the `wendy@pc:~$` line where you type commands.)*

**pwd** *(M4)* — "print working directory": tells you which folder you're standing in.

**Python** *(M5)* — a readable, beginner-friendly programming language, central to AI.

**Repository (repo)** *(M6)* — a shared project on GitHub: its code plus its full history.

**Script** *(M5)* — a text file full of instructions the computer runs (a Python file is a script).

**SSH** *(M8)* — a command that opens a terminal on a remote computer as if it were in front of you.

**subprocess** *(M7)* — a Python tool that lets one program run another and capture its output.

**sudo** *(M2)* — "super-user do": run a command with permission to make powerful changes. Asks your password (typed invisibly).

**Terminal** *(M1)* — the text window where you talk to the computer with words instead of clicks.

**Ubuntu** *(M0)* — the friendly, popular version of Linux that the AI world uses.

**Variable** *(M5)* — a named place to hold a value in code (like `name`).

**WSL (Windows Subsystem for Linux)** *(M1)* — the Microsoft technology that runs a real Ubuntu safely *inside* Windows.
